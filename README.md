## install

-   clone
-   create .env file (can be copy of .env.example)
-   add you mysql info mine was defult
-   create the DB in phpmyadmin
-   composer install
-   php artisan key:generate
-   php artisan migrate
-   npm install
-   npm run dev & php artisan serve (both should be run)
-   opne 127.0.0.1:8000
